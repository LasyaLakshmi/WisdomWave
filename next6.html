19. Generics in Java

What are Generics?

Generics allow you to define classes, interfaces, and methods with type parameters.

Generic Method Example

public class GenericExample {
    public static  void printArray(T[] array) {
        for (T element : array) {
            System.out.println(element);
        }
    }

    public static void main(String[] args) {
        Integer[] intArray = {1, 2, 3};
        printArray(intArray);
    }
}

20. Interfaces in Java

What is an Interface?

An interface is a reference type in Java, it is similar to a class but can contain only constants, method signatures, default methods, static methods, and nested types.

Interface Example

interface Animal {
    void sound();
}

class Dog implements Animal {
    public void sound() {
        System.out.println("Woof!");
    }
}

public class Main {
    public static void main(String[] args) {
        Animal dog = new Dog();
        dog.sound();
    }
}

21. Abstract Classes in Java

What is an Abstract Class?

An abstract class in Java is a class that cannot be instantiated and is used to provide common functionality for subclasses.

Abstract Class Example

abstract class Shape {
    abstract void draw();
}

class Circle extends Shape {
    void draw() {
        System.out.println("Drawing Circle");
    }
}

public class Main {
    public static void main(String[] args) {
        Shape shape = new Circle();
        shape.draw();
    }
}
