C Programming Tutorial

1. Introduction to C

What is C?

C is a general-purpose, procedural programming language developed in the early 1970s by Dennis Ritchie at Bell Labs. It is widely used for system programming, developing operating systems, embedded systems, and more.

Key Features of C

  • Portable and machine-independent
  • Supports structured programming
  • Efficient and fast
  • Rich set of built-in operators and libraries
  • Low-level access to memory through pointers

2. C Syntax Basics

Hello World Program

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

Basic Syntax Rules

  • #include <stdio.h>: Preprocessor directive to include standard input-output functions
  • int main(): Entry point of the program
  • printf(): Function to print output
  • return 0;: End of the program

3. Data Types

Primitive Data Types

  • int - integers (e.g., 1, 2, 3)
  • float - floating point numbers (e.g., 3.14)
  • char - single characters (e.g., 'a')
  • double - double-precision floats

Variable Declaration Example

int age = 25;
double price = 19.99;
printf("Age: %d\n", age);
printf("Price: %.2f\n", price);
