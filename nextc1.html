C Programming Tutorial

5. Variables in C

Variables are used to store data in a C program. Each variable must be declared before it can be used. 

int age = 18;
char grade = 'A';
float height = 5.9;

6. Data Types

  • int - integers (e.g., 1, 2, 3)
  • float - floating point numbers (e.g., 3.14)
  • char - single characters (e.g., 'a')
  • double - double-precision floats

7. Constants

Constants are values that do not change during the execution of a program. 

#define PI 3.14
const int MAX = 100;

8. Keywords

Keywords are reserved words that have special meaning in C. For example:

  • int
  • return
  • if
  • else
  • while
