7. Inheritance

What is Inheritance?

Inheritance allows one class to inherit properties and methods from another class using the extends keyword.

class Animal {
    void eat() {
        System.out.println("This animal eats food.");
    }
}

class Dog extends Animal {
    void bark() {
        System.out.println("Dog barks");
    }
}

public class Test {
    public static void main(String[] args) {
        Dog d = new Dog();
        d.eat();  // Inherited
        d.bark(); // Own method
    }
}

8. Polymorphism

Types of Polymorphism

  • Compile-time (Method Overloading)
  • Runtime (Method Overriding)
// Method Overloading
class MathUtils {
    int add(int a, int b) { return a + b; }
    double add(double a, double b) { return a + b; }
}

// Method Overriding
class Animal {
    void sound() {
        System.out.println("Some sound");
    }
}

class Cat extends Animal {
    void sound() {
        System.out.println("Meow");
    }
}

9. Encapsulation

What is Encapsulation?

Encapsulation is the practice of wrapping data and methods into a single unit (class) and restricting access using access modifiers.

class Person {
    private String name;

    public void setName(String n) {
        name = n;
    }

    public String getName() {
        return name;
    }
}
Note: Use private for fields and public getters/setters for controlled access.

10. Abstraction

What is Abstraction?

Abstraction hides complex implementation details and shows only the essential features.

Implemented using abstract classes and interfaces.

// Abstract class
abstract class Shape {
    abstract void draw();
}

class Circle extends Shape {
    void draw() {
        System.out.println("Drawing a circle");
    }
}
