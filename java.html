Java Programming Tutorial

1. Introduction to Java

What is Java?

Java is a high-level, object-oriented programming language developed by Sun Microsystems (now owned by Oracle). It's designed to be platform-independent, meaning Java code can run on any device that has a Java Virtual Machine (JVM).

Key Features of Java

  • Platform Independent: Write once, run anywhere (WORA)
  • Object-Oriented: Follows OOP concepts like inheritance, polymorphism, etc.
  • Secure: Provides security features like bytecode verification
  • Multithreaded: Supports concurrent programming
  • Robust: Strong memory management and exception handling

2. Java Syntax Basics

Hello World Program

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Basic Syntax Rules

  • Java is case-sensitive
  • Class names should start with uppercase letter
  • Method names should start with lowercase letter
  • Program execution begins with main() method
  • Each statement must end with a semicolon
Note: The filename must match the class name (with .java extension). For the above example, the file must be named "HelloWorld.java".

3. Variables and Data Types

Primitive Data Types

  • byte - 8-bit integer
  • short - 16-bit integer
  • int - 32-bit integer (most common)
  • long - 64-bit integer
  • float - 32-bit floating point
  • double - 64-bit floating point (most common)
  • boolean - true/false values
  • char - single 16-bit Unicode character

Variable Declaration Example

public class Variables {
    public static void main(String[] args) {
        int age = 25;
        double price = 19.99;
        char grade = 'A';
        boolean isJavaFun = true;
        
        System.out.println("Age: " + age);
        System.out.println("Price: " + price);
    }
}
