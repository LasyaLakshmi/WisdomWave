C Programming Tutorial

9. Operators in C

  • Arithmetic: +, -, *, /, %
  • Relational: ==, !=, >, <, >=, <=
  • Logical: &&, ||, !
  • Assignment: =, +=, -=, etc.
int a = 10, b = 5;
int sum = a + b;
int result = (a > b) && (b != 0);

10. Expressions

Expressions are combinations of variables, constants, and operators which are evaluated to produce a value.

11. Type Conversion

Automatic and explicit conversion of data types. 

int a = 10;
double b = a; // implicit
int c = (int)3.14; // explicit (casting)

12. Input/Output

Use scanf to take input and printf to print output. 

#include <stdio.h>
int main() {
  int age;
  printf("Enter your age: ");
  scanf("%d", &age);
  printf("You are %d years old", age);
  return 0;
}
