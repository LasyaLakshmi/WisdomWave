4. Control Flow Statements

Conditional Statements

Java supports decision-making through conditional statements.

  • if, if-else, and else-if
  • switch - for multiple specific value checks
int num = 10;
if (num > 0) {
    System.out.println("Positive");
} else {
    System.out.println("Not Positive");
}

Looping Statements

  • for, while, and do-while loops
for (int i = 1; i <= 3; i++) {
    System.out.println("Count: " + i);
}

5. Methods in Java

Defining and Calling Methods

Methods are reusable blocks of code that perform tasks.

public class MyClass {
    static void greet() {
        System.out.println("Hello!");
    }

    public static void main(String[] args) {
        greet();
    }
}

Method Parameters and Return

static int add(int a, int b) {
    return a + b;
}

6. Object-Oriented Programming Basics

Classes and Objects

A class is a blueprint; an object is an instance of a class.

class Car {
    String color = "Red";

    void drive() {
        System.out.println("Car is driving");
    }
}

public class Main {
    public static void main(String[] args) {
        Car myCar = new Car();
        myCar.drive();
    }
}

Constructors

class Student {
    String name;

    Student(String n) {
        name = n;
    }
}
