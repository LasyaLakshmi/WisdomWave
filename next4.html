13. Java Packages

What are Packages?

Packages in Java are used to group related classes, interfaces, and sub-packages. They help organize code and avoid name conflicts.

Types of Packages

  • Built-in packages: java.util, java.io, etc.
  • User-defined packages: Created by programmers for organizing their own classes.

Creating a Package

package mypackage;

public class MyClass {
    public void display() {
        System.out.println("This is my package!");
    }
}

14. Java Wrapper Classes

What are Wrapper Classes?

Wrapper classes convert primitive data types into objects. They are found in the java.lang package.

Examples

  • int → Integer
  • char → Character
  • boolean → Boolean
int a = 10;
Integer obj = Integer.valueOf(a);  // Boxing
int b = obj.intValue();            // Unboxing

15. Java Exception Handling

What is Exception Handling?

Exception handling in Java deals with runtime errors, helping programs run smoothly without crashing.

Try-Catch Example

try {
    int divide = 10 / 0;
} catch (ArithmeticException e) {
    System.out.println("Cannot divide by zero!");
}

Keywords

  • try, catch, finally, throw, throws
